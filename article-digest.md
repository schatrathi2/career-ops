## What I've built

**Delta Air Lines — Alert Dashboard (Summer 2025)**
The existing supply chain alert system sent emails to hiring managers, but there was no way to track whether anyone had reviewed or resolved the alerts, and no way to quantify whether the system was actually saving money. I talked to the managers to understand what they needed, then designed and deployed a PowerApps application that centralized all alerts in one place. Managers could filter and sort their alerts, mark them as reviewed or resolved, log money saved, and leave comments explaining why something couldn't be resolved. The backend required changes to the Python and SQL pipeline that generated the alerts. The result was that my team could finally follow up with managers who hadn't reviewed their alerts, and could start building a picture of the system's ROI. The project was deployed in my last week, so I don't have outcome metrics — but my manager offered me a full-time role and wanted me to start in January.

**Selero — Prescription Data Cleaning (Summer 2024)**
Processed and cleaned large volumes of messy prescription data using pandas and NLTK, enabling the audit team to detect unsubmitted 340B claims. This was mostly unglamorous data wrangling work, but the output had real compliance and financial impact.

**Toxic Spans Detection — SemEval 2021 Task 5**
Built a token-frequency pipeline for character-level toxic span detection in text, evaluated using the official SemEval F1 metric. GitHub: https://github.com/schatrathi2/toxic-spans-detection

**Job Sentiment Analysis**
NLP pipeline for sentiment analysis on job postings. GitHub: https://github.com/schatrathi2/job-sentiment-analysis

**Research — ChatGPT and Older Adults (Spring 2025)**
Conducted statistical analysis on a dataset about older adults' use of ChatGPT to surface behavior patterns and usability gaps. Cleaned and transformed the data using pandas and NumPy.
